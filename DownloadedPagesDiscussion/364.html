Ahem. So, after some discussion, we have a proposed (and accepted by several players) solution to the sign up dilemma.

Rather than use a system like Umbra's, we can just add an RP Sample section onto the end of the signup sheet. There would be a prompting post (probably the same for each, almost definitely a battle of some kind) that the signer-upper responds to in this section.

Using this method, we can vet out crimes against grammar/spelling, make note of godmodders/powerplayers/etc., and ask them to improve before allowing them in.

Presumably, the RP Sample would be allowed to be edited to allow for improvements.

Sound good? I think it's much simpler.

Tangentially related: I think we should make a proper sign up thread.
