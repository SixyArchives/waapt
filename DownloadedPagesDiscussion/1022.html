Since I think it's at an okay point for general WAAPT use, I'll go ahead and show this thing now.

THE WAAPT SEARCH

What is it?

A system that lets you record information like author, characters, arc, date, and plot for any posts made in WAAPT (the main thread in addition to canon WAAPT Forum Threads) and search for posts that meet certain criteria. It's still going to have things added to it later, but I doubt there are going to be any big problems with it at the moment.

Why's it asking for a username/password?

Tangent put that in as a security measure, but it'll eventually only ask for those when you're trying to edit information. I can send them to you via PM/IM if you ask.

There aren't many posts showing up in the search...

Yeah, there are quite a few posts without any data. We'll see if we can't get some automation running later, but I'm hoping people will help some by adding data for their own/other's posts. Granted, I could end up being the only one adding info, but I'd appreciate it if others would too.

I made an edit earlier and it disappeared a second later

I hope this doesn't happen, but if more than one person tries editing the same thing at once, it might. Tell me if it does, so we can hopefully get some way of handling it in.

Why's it not letting me edit in a new post?

Sorry about that. With how it works currently, if you try and edit a post whose number doesn't exist in the data, it won't edit it in. You'll need to update the number of posts in the thread first with one of the 'Add posts to thread' links.

Why isn't the text showing in the summaries for posts?

Text isn't really displaying neatly right now and there isn't a search for text yet, so it doesn't do much. We'll see if we can't get that fixed later.

Why is X Thread in/not in the drop-down list?

I tried including all the Canon WAAPT forum threads I could, but I might have forgotten/misclassified one or two. If that's the case, tell me so I can fix it?

What's the 'Commit' Button do?

That's mostly for version control of the post data on my end. You're free to ignore it since Editing automatically saves it, but if you want to press it after an edit, feel free.

Could you add X feature/improve Y thing about the website?

I can try. Just make sure to tell me.

What are some things you're planning on adding yet?

  • The number of results given by Search
  • Searching across a range of dates
  • Something that generates a full list of characters/locations/arcs/plots/whatever else

So, yeah. Hope you guys find this interesting and make use of it. Make sure to give me any feedback you might have on it.
