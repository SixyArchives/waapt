It'd make far more sense if a sole name was agreed upon for each line. Otherwise, you'd run into that problem with anybody who takes taxonomy seriously. A trinomial name works far better, specially when you get to deal with the extremelly similar Pokemon; you'd want to make sure that Vulpix is closer to Ninetales that to Zorua, but, at the same time, that all 3 of them are closer to each other that to Growlithe.

Trinomial names work better; close Pokemon share only the first one (genus), families have the same second as well (species), but different stages also differ in their third name (subspecific/morph/whatever).
